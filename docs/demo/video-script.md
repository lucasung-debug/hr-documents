# 포트폴리오 데모 영상 상세 촬영 대본

> 대상: 비개발자 HR 담당자 / HR SaaS 기업 채용 담당자  
> 용도: 포트폴리오 임베드 및 면접 시연  
> 기획 원본: `docs/plans/2026-05-24-portfolio-video-enhancement.md`, `docs/plans/2026-05-24-council-report.md`  
> ⚠️ 법적 표기 준수: 본 영상은 **"전자서명법 법적효력"을 주장하지 않으며**, **"서명 무결성 검증(SHA-256 변조 감지)"** 기능만을 설명합니다.

---

## 1. 영상 개요

| 항목 | 내용 |
|------|------|
| 풀 버전 | 2분 30초 (150초) |
| 포트폴리오 임베드 컷 | 60초 |
| 해상도 | 1920×1080 (16:9) 또는 1440×900 (녹화 최적화) |
| 프레임레이트 | 30fps |
| 음성 | 남성/여성 무관, 침착한 속도 (분당 260~280어절), 선택적 자막 병행 |
| 자막 규칙 | 화면 중앙 하단, 고딕계 서체, 흰색+검정 아웃라인, 최소 48pt |
